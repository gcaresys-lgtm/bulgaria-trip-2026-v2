# Bulgaria 2026 - Sunny Beach Vacation PWA

Professional travel companion application built with React + TypeScript.

## Tech Stack

- React 18 + TypeScript
- Zustand for state management
- Tailwind CSS 4
- Framer Motion for animations
- Lucide React icons
- PWA with Service Worker

## Features

- 🏠 Home Dashboard with countdown and quick actions
- 📅 Interactive daily itinerary
- 📸 Photo gallery with compression
- 💡 Travel tips and information
- 🎒 Packing checklist with progress
- 🚨 Emergency contacts and hotel info
- 💱 Currency converter
- 📴 Full offline support

## Development

```bash
npm install --legacy-peer-deps
npm run dev
```

## Build

```bash
npm run build
```
